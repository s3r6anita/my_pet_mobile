## Quality Gate. Работа и настройка

Локальный запуск утилиты `detekt` для проверки качества кода осуществляется с помощью 
последовательного исполнения следующих команд:
```
chmod +x detekt
./detekt 
```
В случае успешного выполнения выводится `Success. Not found errors`, в противно случае в консоль 
выводится отчет с описанием проблем и мест, где они обнаружены. 

Правила detekt (конфиг) расположены в файле [.github/workflows/assets/detekt.yml](.github/workflows/assets/detekt.yml)
